# AI Tab Clear

[中文文档](USER_MANUAL.md) | [English](README_EN.md)

**AI Tab Clear** 是一款智能的 Chrome 浏览器标签页管理扩展，通过自动关闭闲置标签页和智能分类历史记录，帮助您保持浏览器的整洁与高效。

## ✨ 核心特性

- **🧠 智能分组**：支持 **AI (Gemini/OpenAI)** 和 **本地算法** 两种方式对历史记录进行智能分类。
- **⚡️ 自动清理**：根据闲置时间和标签页数量阈值，自动关闭不活动的标签页。
- **🌍 多语言支持**：内置 9 种语言（中、英、日、韩、德、法、西、意），一键切换。
- **🎨 现代化 UI**：精美的卡片式设计，支持深色模式和玻璃态特效。
- **🔍 历史搜索**：支持实时搜索和过滤已关闭的标签页，支持一键恢复。
- **🛡️ 灵活规则**：支持白名单、黑名单和固定标签页保护。

## 🚀 快速开始

1. **安装**：加载已解压的扩展程序（开发者模式）。
2. **配置**：点击图标进入设置，调整闲置时间和分组方式。
3. **使用**：扩展将在后台自动运行。点击图标可查看历史记录、搜索或手动管理。
