# AI Tab Clear - 快速开始指南

## 🚀 立即安装

### 步骤 1: 打开扩展管理页面
在 Chrome 浏览器地址栏输入:
```
chrome://extensions/
```

### 步骤 2: 启用开发者模式
点击页面右上角的"开发者模式"开关

### 步骤 3: 加载扩展
1. 点击"加载已解压的扩展程序"按钮
2. 选择文件夹: `/Users/kaifengyang/Desktop/antigravity/ai-tab-clear`
3. 点击"选择"

### 步骤 4: 验证安装
- ✅ 扩展列表中出现 "AI Tab Clear"
- ✅ 工具栏显示扩展图标
- ✅ 点击图标显示弹窗界面

## ⚙️ 快速配置

### 基础设置 (推荐)
1. 点击扩展图标打开弹窗
2. 切换到"设置"标签页
3. 设置闲置时间: **30 分钟** (默认)
4. 设置最小标签页数量: **10 个** (默认，超过此数量才开始自动关闭)
5. 点击"保存设置"
6. 点击右上角 🌙 图标可切换到日间模式

### 白名单配置 (可选)
在白名单中添加常用网站，防止被关闭:
```
google\.com
github\.com
localhost
```

### 黑名单配置 (可选)
在黑名单中添加想要立即关闭的网站:
```
ads\..*
spam\..*
```

## 🧪 快速测试

### 测试自动关闭 (1分钟快速测试)
1. 将闲置时间设置为 **1 分钟**
2. 打开几个测试标签页 (如 example.com)
3. 不要点击这些标签页
4. 等待 1 分钟
5. ✅ 标签页应该自动关闭

### 测试历史记录
1. 标签页关闭后，点击扩展图标
2. 查看"历史记录"标签页
3. ✅ 应该看到刚才关闭的标签页
4. 点击任意历史记录
5. ✅ 标签页应该重新打开

### 测试暂停功能
1. 点击右上角"暂停"按钮
2. ✅ 按钮变为绿色，显示"继续"
3. 等待超过闲置时间
4. ✅ 标签页不应该关闭
5. 点击"继续"按钮恢复功能

## 🎯 使用技巧

### 正则表达式示例
```
# 匹配特定域名
google\.com

# 匹配子域名
.*\.github\.com

# 匹配多个域名
(google|github|stackoverflow)\.com

# 匹配本地开发
localhost|127\.0\.0\.1

# 匹配所有 .cn 域名
.*\.cn
```

### 推荐设置
- **日常使用**: 闲置时间 30-60 分钟
- **工作场景**: 闲置时间 15-30 分钟
- **演示/展示**: 暂停功能

## ❓ 常见问题

**Q: 标签页没有自动关闭?**
- 检查是否处于暂停状态
- 确认标签页不在白名单中
- 验证闲置时间设置

**Q: 如何防止重要网站被关闭?**
- 在白名单中添加对应域名的正则表达式

**Q: 历史记录会保存多久?**
- 最多保存 100 条记录，超过后自动删除最旧的
