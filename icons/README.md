# Icon Placeholder

Due to image generation quota limits, please add icon files manually:

## Required Icons

Create three PNG icon files in this directory:
- `icon16.png` (16x16 pixels)
- `icon48.png` (48x48 pixels)  
- `icon128.png` (128x128 pixels)

## Design Recommendations

- **Style**: Modern, minimalist flat design
- **Colors**: Purple to blue gradient (#6366f1 to #8b5cf6)
- **Symbol**: Tab icon with a subtle "X" or close indicator
- **Background**: Transparent or dark

## Quick Icon Creation Tools

You can use these online tools to create icons quickly:
- [Favicon.io](https://favicon.io/) - Simple icon generator
- [Canva](https://www.canva.com/) - Design tool with templates
- [Figma](https://www.figma.com/) - Professional design tool

## Temporary Solution

If you want to test the extension immediately without custom icons, you can:
1. Create simple colored squares as placeholders
2. Use any existing PNG images (will be resized automatically)
3. Download free icons from [Flaticon](https://www.flaticon.com/) or [Icons8](https://icons8.com/)
